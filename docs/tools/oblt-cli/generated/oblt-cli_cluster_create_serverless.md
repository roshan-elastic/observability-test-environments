## oblt-cli cluster create serverless

Command to create an serverless cluster.

### Synopsis

Command to create a custom cluster using the serverless template. If used, any of the docker images must use the same version as the stack version, otherwise the build of the cluster will fail in Elastic Cloud

```
oblt-cli cluster create serverless [flags]
```
