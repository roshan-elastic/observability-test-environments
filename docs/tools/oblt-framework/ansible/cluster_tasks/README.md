# Cluster Tasks Role

## Overview

The Cluster Tasks role is a collection of tasks that are used to manage the cluster. The tasks are used to build the cluster, deploy the cluster, and manage the cluster.

## Requirements

It uses the oblt framework roles.

## Parameters

### Cluster Tasks

| Name            | Description                                  | Value |
| --------------- | -------------------------------------------- | ----- |
| `templates_dir` | The directory where the templates are stored | `""`  |
| `build_dir`     | The directory where the build is stored      | `""`  |
