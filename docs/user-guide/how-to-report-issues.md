## How to report an issue

When you face an error with oblt-cli, you can report the issue by creating a new issue in the [oblt-cli repository](https://github.com/elastic/observability-test-environments/issues/new?assignees=&labels=bug%2C+area%3Aclusters%2C+team%3AAutomation%2C+triage&projects=&template=incident-bug-report.md&title=%5BIncident%5D+).
For questions and support you can ask in the [#observablt-robots](https://elastic.slack.com/archives/CJMURHEHX) channel in Slack.
In order to troubleshoot your issue, we will need the maximum information you can provide, such as the command you ran, the output of the command, the environment you are using, and the error message.
To start, you can use the `--verbose` flag to get more information about the error.
