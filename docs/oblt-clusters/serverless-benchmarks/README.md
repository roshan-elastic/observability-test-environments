# Serverless Benchmarks

## Overview

This Serverless project is similar to [serverless-qa-oblt][], but without apps deployed.
edge-benchmarks is used to run the benchmarks for the edge versions.

## Update

The update of the clusters is automatic when the [QA environment][] push a new version of the stack.
The version to update is the latest stable version of the latest on the [QA environment][].

[QA environment]: https://docs.elastic.dev/serverless/qa
